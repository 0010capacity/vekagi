# VANGUARD PUSH — UI/UX & 화면 흐름

## 화면 전환 흐름

```
main_menu → piece_pick → run_map → formation_setup → battle → reward → run_map (반복)
                                                              ↓
                                                           run_end (패배/클리어)
```

## UX 원칙

### 충돌 미리보기
- 기물 선택 → 이동 가능 칸 파란색 하이라이트
- 각 칸 hover → 밀려나는 칸 수 숫자 표시
- 연쇄 경로 점선 표시

### 적 행동 의도
- 모든 적 기물 위에 이동 방향 화살표 (주황색)
- 클릭 시 시뮬레이션 팝업

### 연쇄 충돌 연출
- 0.3초 간격 순차 연출
- 충돌 지점 임팩트 이펙트
- 전달 힘 수치 표시
