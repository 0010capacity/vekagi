# VANGUARD PUSH — 게임 컨셉 & 규칙

## 핵심 컨셉

VANGUARD PUSH는 그리드 위에서 펼쳐지는 **물리 기반 전략 로그라이크** 게임이다.
플레이어는 고유한 이동 방식과 스탯을 가진 기물들을 조합하여 진형을 구성하고,
매 층마다 적 기물을 그리드 밖으로 밀어내어 전멸시키는 것이 목표다.

**레퍼런스:** Into the Breach + Slay the Spire + 체스 + 물리 충돌 퍼즐

## 충돌 시스템

### 기본 공식
```
밀려나는 칸 수 = 1 (기본) + max(0, 공격자 힘 − 피격자 질량)
```

- 질량이 아무리 높아도 최소 1칸은 밀린다
- 힘이 질량보다 클수록 더 멀리 밀린다
- 플레이어는 항상 정확하게 몇 칸 밀릴지 계산 가능

| 공격자 힘 | 피격자 질량 | 계산 | 결과 |
|---|---|---|---|
| 1 | 5 | 1 + max(0, -4) | 1칸 |
| 3 | 3 | 1 + max(0, 0) | 1칸 |
| 4 | 2 | 1 + max(0, 2) | 3칸 |
| 5 | 1 | 1 + max(0, 4) | 5칸 |
| 5 | 5 | 1 + max(0, 0) | 1칸 |

### 연쇄 충돌
```
전달 힘 = 공격자 힘 − 피격자 질량
전달 힘 > 0이면 연쇄 발생 / 0 이하면 연쇄 종료
```

**예시: A(힘5) → B(질량2) → C(질량3)**
1. A가 B 충돌: 밀림 = 1+3 = 4칸, 전달 힘 = 5−2 = 3 → 연쇄 발생
2. B가 C 충돌: 밀림 = 1+0 = 1칸, 전달 힘 = 3−3 = 0 → 연쇄 종료

> **관통 특성:** 감쇠 없이 원래 힘 그대로 전달

## 턴 구조

```
① 계획 페이즈
   - 플레이어: AP 소모하여 기물 이동 예약
   - 적 AI: 이번 턴 행동 의도(이동 방향) 화살표로 공개

② 해결 페이즈
   - 모든 기물이 d6 + 민첩으로 우선권 결정
   - 결과 공개 후 실행 순서 확정

③ 실행 페이즈
   - 우선권 높은 순서대로 이동 실행
   - 충돌 발생 시 공식에 따라 즉시 계산
   - 연쇄 충돌 순차 처리

④ 카운트다운 페이즈
   - 카운트다운 기물 타이머 1 감소
   - 타이머 0 도달 시 해당 효과 발동
```

## 행동력 (AP) 시스템

- 플레이어 턴당 AP: **6**
- 각 기물 AP 비용: **1~3**
- 미사용 AP는 이월되지 않는다
- 기물 6개 보유 시 전부 조작하려면 AP가 부족한 경우 발생 → 매 턴 '어떤 기물을 포기할 것인가' 의사결정

---

## 지휘관 HP 시스템

기물은 HP가 없다. 기물이 그리드 밖으로 밀려나거나 위험 타일에 닿으면 즉시 사망.
기물 사망 시 **지휘관 HP**가 감소한다.

| 상황 | HP 변화 |
|---|---|
| 아군 기물 사망 (기본) | −1 |
| 양날의검 특성 기물 사망 | −2 |
| 무른 특성 기물 사망 | ±0 |
| 순교자 특성 기물 사망 | +1 |
| 지휘관 HP 0 | 런 종료 (패배) |

**핵심 설계 의도:**
- 기물은 영구 소실되지 않음 → 사망한 기물은 다음 층 시작 시 1HP(최저 스탯)로 부활
- 희생 전략을 자유롭게 쓸 수 있되, 그 대가는 런 전체에 누적됨
- Slay the Spire처럼 매 층 최소 손실로 클리어하는 것이 핵심

## 기물 배치 규칙

- 플레이어 최대 보유 기물: **6개**
- 런 시작 시 3기물 선택
- 구역 클리어마다 슬롯 +1 (최대 6)
- 각 층 시작 전 진형 카드를 선택하고 기물을 슬롯에 배치

## AI 설계

### 적 행동 공개 원칙
모든 적 기물은 해결 페이즈 전에 이번 턴 이동 의도(방향 화살표)를 공개한다.
플레이어는 이를 보고 대응 전략을 수정할 수 있다.

### 적 행동 원형 (Archetype)
| 원형 | 행동 패턴 |
|---|---|
| 공격형 | 가장 가까운 플레이어 기물을 향해 이동, 밀어내기 우선 |
| 압박형 | 플레이어 기물을 그리드 가장자리 방향으로 유도 |
| 보호형 | 아군 기물 인접에 머물며 버프 유지 |
| 자폭형 | 카운트다운 기물. 플레이어 진형 중심을 향해 이동 |
| 소환형 | 고정 위치 유지하며 매 턴 기물 생성 |
| 혼란형 | 랜덤 이동. 예측이 가장 어려움 |

### AI 구현 로드맵
- **Phase 1 (출시):** 룰 기반 AI — 행동 우선순위 트리 (즉시 제거 → 위험 타일 유도 → 연쇄 → 가장자리 몰기 → 접근)
- **Phase 2 (업데이트):** MCTS + 경량 신경망 (TensorFlow.js, Web Worker 분리 실행)

### AI 보상 함수 (Phase 2)
```
+10  플레이어 기물 그리드 밖으로 밀어냄
+15  플레이어 기물을 위험 타일로 밀어냄
+3   연쇄 충돌 발생 (연쇄당)
+1   플레이어 기물을 가장자리로 유도 (인접당)
-10  아군 기물이 그리드 밖으로 밀려남
-15  아군 기물이 위험 타일에 닿음
-0.5 턴 소비 페널티
```
