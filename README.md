# Vekagi (베카기)

> 그리드 위에서 펼쳐지는 물리 기반 전략 로그라이크 게임

**Into the Breach × Slay the Spire × 체스 × 물리 충돌 퍼즐**

---

## 🎮 게임 소개

Vekagi는 독특한 **물리 충돌 시스템**을 핵심으로 한 전략 로그라이크입니다. 고유한 이동 방식과 스탯을 가진 기물들을 조합하여 진형을 구성하고, 매 층마다 적 기물을 그리드 밖으로 밀어내어 전멸시키세요.

### 핵심 특징

- **물리 충돌 시스템** — 힘, 질량, 연쇄 충돌을 계산하여 적을 그리드 밖으로 밀어냅니다
- **50종 이상의 기물** — 각기 다른 이동 방식, 스탯, 특성을 가진 기물을 수집하고 조합하세요
- **24종 진형 카드** — 기물 배치에 따라 발동되는 시너지 효과
- **적 행동 공개** — 적 AI의 이동 의도를 미리 확인하고 대응 전략을 세우세요
- **로그라이크 구조** — 매 런마다 새로운 기물 조합과 진형을 실험하세요

## 🕹️ 충돌 시스템

```
밀려나는 칸 수 = 1 (기본) + max(0, 공격자 힘 − 피격자 질량)
```

| 공격자 힘 | 피격자 질량 | 결과 |
|:---:|:---:|:---:|
| 1 | 5 | 1칸 밀림 |
| 3 | 3 | 1칸 밀림 |
| 4 | 2 | 3칸 밀림 |
| 5 | 1 | 5칸 밀림 |

연쇄 충돌이 발생하면 밀려난 기물이 또 다른 기물과 충돌합니다. 도미노처럼 한 번의 행동이 연쇄적인 전멸을 일으킬 수 있죠.

---

## 🎯 턴 구조

1. **계획 페이즈** — AP를 소모하여 기물 이동을 예약합니다
2. **해결 페이즈** — 모든 기물이 d6 + 민첩으로 우선권을 결정합니다
3. **실행 페이즈** — 우선권 순서대로 이동이 실행되고 충돌이 계산됩니다
4. **카운트다운 페이즈** — 타이머 기물의 효과가 발동됩니다

## 📁 프로젝트 구조

```
src/
├── components/       # UI 컴포넌트
│   ├── board/        # 게임 보드, 셀, 기물 토큰
│   ├── pieces/       # 기물 카드, 스탯, 특성 배지
│   ├── formations/   # 진형 카드
│   ├── hud/          # AP바, HP바, 턴 순서
│   ├── map/          # 런 맵
│   └── modals/       # 상점, 이벤트, 보스 모달
├── engine/           # 충돌 엔진, 턴 엔진
├── stores/           # 게임/런/UI 상태 스토어
├── workers/          # AI Web Worker
├── data/             # 기물, 진형, 보스, 이벤트 데이터
├── hooks/            # 게임 엔진 훅, AI 훅
├── screens/          # 화면 (메인메뉴, 전투, 맵 등)
└── types/            # TypeScript 타입 정의
```
